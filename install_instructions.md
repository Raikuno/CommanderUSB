## Instrucciones de instalación
A la hora de realizar las instalaciones se deberá de seguir un proceso distinto para la aplicación del servidor que para la aplicación de cliente.

Para la **aplicación de servidor** bastará con simplemente iniciar la aplicación una vez se hayan instalado las dependencias necesarias, encargandose la 
propia aplicación de inicializar la base de datos en local si esta no existe ya. En caso de necesitarlo, se podrá cambiar el puerto en el que la aplicación se ejecuta, asi 
como otros parámetros importantes modificando los campos pertinentes en el archivo de configuración.

Por otro lado, la instalación de la **aplicación de cliente** será muy similar a la aplicación de servidor, siendo que solo será necesario tener en cuenta que:
- La IP y Puerto a introducir en la aplicación deben corresponder con aquellos en los que este funcionando la aplicación de servidor.
- La aplicación deberá ejecutarse con permisos de administrador para asegurar que esta pueda configurarse para ser lanzada al iniciar el sistema operativo
